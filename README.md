
## Setup


1. Install the [bundler gem](http://bundler.io/) by running:

    ```gem install bundler```

2. Clone this repo:

    ```git clone git@github.com:ThiagoHocsis/toy_robot_simulator.git```

3. Change to the app directory:

    ```cd toy_robot_simulator```

4. Install dependencies:

    ```bundle install```

### Running the app:
```ruby lib/main.rb```

### Running the tests:
```bundle exec rspec```
